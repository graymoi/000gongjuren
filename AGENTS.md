# Agents 配置

## 文档处理代理 (Document Processor Agent)

### 功能
- 处理 Word 文档（.docx）
- 处理 Excel 表格（.xlsx）
- 处理 PDF 文件（.pdf）
- 文档格式转换
- 内容提取和整理

### 触发条件
- 上传 .docx, .xlsx, .pdf 文件
- 用户请求文档处理任务

### 工作流程
1. 识别文件类型
2. 分析文档内容
3. 执行用户指定的操作
4. 生成输出文件

## 图片处理代理 (Image Processor Agent)

### 功能
- 图片格式转换
- 图片压缩
- 图片尺寸调整
- 批量图片处理

### 触发条件
- 上传图片文件（.jpg, .png, .bmp, .gif 等）
- 用户请求图片处理任务

### 工作流程
1. 分析图片属性
2. 执行用户指定的操作
3. 生成输出文件

## 投标文件整理代理 (Bid Document Agent)

### 功能
- 整理投标文件
- 组合拼装文档
- 生成投标书
- 检查文件完整性

### 触发条件
- 用户上传投标相关文件
- 用户请求投标文件整理

### 工作流程
1. 分析投标文件结构
2. 整理文件顺序
3. 组合拼装文档
4. 生成最终投标书
